### bb tooling caveat

Standalone `bb gates -b <acir>` defaults to `ultra_honk` and rejects aztec private circuits with `UltraCircuitBuilder does not support CallData/ReturnData block constraints. Use MegaCircuitBuilder (Aztec app) or fall back to RAM and ROM operations.` Workaround: pass `-s chonk`. The aztec docker image bundles a Mega-capable bb (likely `bb_aztec`); standalone bb with `-s chonk` is a close-enough approximation for gate counting.
